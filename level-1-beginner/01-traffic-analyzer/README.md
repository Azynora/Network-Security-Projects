# 📡 01 - Traffic Analyzer

## Mô tả
Phân tích file `.pcap`, thống kê traffic và phát hiện các dấu hiệu tấn công đơn giản.

## Tính năng dự kiến
- [ ] Parse file .pcap bằng Scapy / Tshark
- [ ] Thống kê: Top talkers, Top protocols, Top ports
- [ ] Phát hiện: SYN flood, Ping sweep, Port scan
- [ ] Xuất báo cáo dạng bảng (rich) và CSV

## Kỹ năng
`Python` `Scapy` `TCP/IP` `Wireshark`
