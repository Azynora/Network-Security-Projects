# 🔍 02 - Network Scanner & Asset Discovery

## Mô tả
Quét mạng LAN, phát hiện thiết bị online và nhận diện OS.

## Tính năng dự kiến
- [ ] ARP scan phát hiện thiết bị trong LAN
- [ ] Port scan các thiết bị tìm được
- [ ] Nhận diện OS qua TTL fingerprint
- [ ] Xuất báo cáo: IP, MAC, Hostname, Open Ports, Vendor

## Kỹ năng
`Python` `Nmap` `ARP` `Socket`
